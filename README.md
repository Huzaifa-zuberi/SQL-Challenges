# SQL Database Design Challenges

[![License](https://img.shields.io/badge/License-MIT-6ef7c8)](LICENSE)

A collection of SQL database design challenges demonstrating relational schema design, complex queries, and data analysis.

## 📂 Challenges

| File | Description |
|------|-------------|
| [Challenge A: Smart Parking](ChallengeA_SmartParking.sql) | Database schema for a smart parking system with real-time availability tracking |
| [Challenge B: E-Commerce Inventory](ChallengeB_ECommerceInventory.sql) | Inventory management system with product categories, stock levels, and order tracking |
| [Challenge C: Churn Telemetry](ChallengeC_ChurnTelemetry.sql) | Telco customer churn analysis with telemetry data and predictive indicators |

## 🛠️ Skills Demonstrated

- Relational database schema design
- SQL Server / MySQL query optimization
- Complex JOINs, subqueries, and aggregations
- Stored procedures and views
- Data integrity constraints (PK, FK, CHECK)
